# 💫 FinanceFlow - Sistema de Gestão Financeira Pessoal

![FinanceFlow](https://img.shields.io/badge/FinanceFlow-Personal%20Finance-blue?style=for-the-badge&logo=bank&logoColor=white)
![React](https://img.shields.io/badge/React-18.2.0-61DAFB?style=for-the-badge&logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-3178C6?style=for-the-badge&logo=typescript)

Um aplicativo web moderno e intuitivo para controle financeiro pessoal, desenvolvido com React, TypeScript e Tailwind CSS.

## 🚀 Funcionalidades Principais

### 📊 **Dashboard Completo**
- **Visão geral financeira** com saldo disponível, contas e projeções
- **Saldo Futuro** para planejamento de receitas esperadas
- **Gráficos interativos** de despesas por categoria
- **Histórico de pagamentos** com status em tempo real

### 💰 **Gestão de Contas**
- ✅ Adicionar, editar e excluir contas
- 📅 Controle de datas de vencimento
- 🏷️ Categorização automática
- 🔔 Sistema de notificações

### 📈 **Analytics Avançados**
- 📊 Gráfico de pizza para categorias de gastos
- 📈 Análise de gastos mensais
- 🎯 Metas financeiras personalizadas
- 💡 Insights inteligentes

### 🗓️ **Calendário Integrado**
- 📅 Visualização mensal de contas e lembretes
- ⏰ Lembretes personalizáveis
- 🔄 Sincronização com contas

### 🔔 **Sistema de Notificações**
- 💸 Alertas de contas próximas do vencimento
- 📱 Notificações push
- 🎯 Lembretes personalizados

### 🎨 **Experiência do Usuário**
- 🌙 **Modo escuro/claro**
- 📱 **Design totalmente responsivo**
- ⚡ **Animações suaves** com Framer Motion
- 🎯 **Interface intuitiva** e moderna

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React 18, TypeScript, Tailwind CSS
- **Animações:** Framer Motion
- **Ícones:** Lucide React
- **Armazenamento:** LocalStorage
- **Build:** Vite

## 📦 Instalação e Uso

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/financeflow.git

# Entre no diretório
cd financeflow

# Instale as dependências
npm install

# Execute o projeto
npm run dev
